Installation Instructions for [Add-on Name]

To install this add-on, please follow these steps:

  1. Open Your Files:
    • Navigate to your NG Extension Manager installation folder using File Explorer (Windows) or Finder (Mac).
  2. Find the Add-ons Folder:
    • Inside the NG Extension Manager folder, locate and open the "addons" folder (or it might still be named "plugins" depending on the manager version - look for where other add-ons are).
    • *If an "addons" folder doesn't exist, you might need to create it.*
  3. Make a New Add-on Folder:
    • Inside the "addons" (or "plugins") folder, create a new, uniquely named folder for this add-on (e.g., `my-cool-addon`).
  4. Get the Add-on Files:
    • Click the "Go to Add-on" button below to open its GitHub page.
    • On the GitHub page, find and download the `addon.json` and `addon.js` (or similar `.js`) files.
    • *Note: File names might vary slightly, but look for a `.json` configuration file and the main JavaScript file.*
  5. Put the Files in Your New Folder:
    • Move the downloaded `addon.json` and `.js` files into the new folder you created (e.g., into `my-cool-addon`).
  6. Reload the NG Extension Manager:
    • Close and reopen the NG Extension Manager application to load the new add-on.

Troubleshooting:

  • If the add-on doesn't appear, double-check the folder name is unique and that both the `addon.json` and `.js` files are correctly placed inside it.
  • Ensure the `addon.json` file has the correct format and the `.js` file is not corrupted.
  • Check the NG Extension Manager's documentation or community forums for further assistance.
